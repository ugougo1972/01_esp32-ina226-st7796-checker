## INA226 学習ルール

本プロジェクトでは INA226 を使用した電流・電圧・電力測定を行うが、ライブラリを単に使用するのではなく、以下の内部動作を理解したうえで使用する。

### 理解対象
- シャント抵抗の役割
- シャント電圧から電流を求める原理（オームの法則）
- INA226 の測定項目

  - Bus Voltage
  - Current
  - Power
- Current LSB の意味
- Calibration Register の意味
- 最大電流設定と分解能の関係

### INA226 設計前提
- 使用シャント抵抗：0.002Ω
- 想定最大電流：6A（USB-C 100W測定を想定）
- 電流分解能：0.2mA/bit を目安
- INA226 はハイサイド電流測定として使用する

### 配線方針
INA226 は電源ラインのプラス側に直列に挿入する（ハイサイド測定）。

## INA226 電流測定に関する設計ルール

本プロジェクトでは INA226 を使用して電流・電圧・電力測定を行う。
ライブラリを使用するが、内部計算の意味を理解した上で使用することを前提とする。

### INA226 の測定原理
INA226 は電流を直接測定するのではなく、シャント抵抗の両端に発生する電圧（シャント電圧）を測定し、オームの法則により電流を算出する。

電流 = シャント電圧 / シャント抵抗

電力 = 電圧 × 電流

### INA226 内部計算
INA226 内部では以下の計算が行われている。

Current = Current_Register × Current_LSB  
Power = Power_Register × Power_LSB  
Power_LSB = 25 × Current_LSB  

また、Calibration Register によりシャント抵抗値と電流スケールを設定する必要がある。

### Current LSB の意味
Current LSB とは「Current Register が 1 増えたときの電流値（最小分解能）」を意味する。

LSB = Least Significant Bit（最下位ビット）であり、本プロジェクトでは「測定分解能」を意味する。

### 本プロジェクトの INA226 設計値
| 項目 | 値 |
|------|----|
| シャント抵抗 | 0.002Ω |
| 想定最大電流 | 6A |
| Current_LSB | 約0.0002A |
| Calibration | 約12800 |

### 測定方式
INA226 はハイサイド電流測定として使用する。

電流の流れ：
電源 + → INA226 → 負荷 → GND → 電源 -

INA226 は電源ラインのプラス側に直列に挿入する。
電流の流れ：
電源 + → INA226 → 負荷 → GND → 電源 -

この順序を間違えないこと。

## [ACTIVE] プロジェクト目的（確定）

本プロジェクトの目的は、ESP32 を用いて USB-C 電源ラインの電圧・電流・電力を測定し、
表示・記録・グラフ化を行う「USB電力ロガー」を製作することである。

単なる表示装置ではなく、以下の機能を持つ計測・記録装置とする。

### 機能要件
- 電圧表示
- 電流表示
- 電力表示
- 積算電力量（Wh）
- 積算電流量（Ah）
- 動作時間計測
- 最大値・平均値表示
- 1時間グラフ表示（電圧・電流・電力）
- microSDカードへのCSV保存
- PCへのシリアルログ出力
- RTCによる時刻管理
- ロータリーエンコーダによるUI操作

### 測定仕様
- 最大測定電圧：20V
- 最大測定電流：5A
- 最大測定電力：100W
- 測定方式：INA226（シャント抵抗方式）

### 記録仕様
- ログ周期：1秒
- グラフ：60秒平均、60点（1時間）
- 保存形式：CSV

### 開発方針
- 段階的に機能を追加する
- 最小構成で動作確認 → 機能追加
- ライブラリ任せにせず、内部動作を理解する
- 測定精度と積算精度を重視する
